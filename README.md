# The 4 seasons
In **Nigeria**, there are four seasons:
+ Rainy Season
+ Harmattan Season
+ Hot Season 1
+ Hot Season 2

These equate to the four seasons in the western world, which are:
1. Summer
2. Autumn
3. Winter
4. Spring
